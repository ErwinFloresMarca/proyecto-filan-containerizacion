# Task Manager - Microservices

## Quick Start

```bash
# Start all services
./start.sh

# Or using make
make start
```

## Services

- Frontend: http://localhost:5173
- Task Service: http://localhost:3001
- User Service: http://localhost:3002
- Prometheus: http://localhost:9090
- Grafana: http://localhost:3005
